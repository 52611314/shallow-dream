## 1.1：环境配置（2026.5.21）

难得这种三分钟热度还能撑到第二天，找DeepSeek要了开发计划，感觉挺宽松的，就先把第一步环境配好吧。

- 创建项目目录 `shallow-dream`，按 `backend/` 结构组织后端代码
- 在 `backend/` 内重建 Python 虚拟环境并安装 FastAPI、LangChain、数据库驱动等依赖
- 编写 `config.py` 使用 `dotenv` 加载环境变量，`main.py` 实现 `/` 和 `/health` 探活接口
- 通过 `uvicorn` 启动服务，验证接口正常响应，排除 `favicon.ico` 404 干扰
- 明确每日开发启动流程：`cd backend` → 激活 venv → `uvicorn app.main:app --reload`
